JSplitPolygon is a JAVA library designed to split a polygon into two polygons when the polygon crosses the antimeridian.

In the GeoJson requirement, the polygon must be cut when it crosses the antimeridian: In representing Features that cross the antimeridian, interoperability is improved by modifying their geometry. Any geometry that crosses the antimeridian SHOULD be represented by cutting it in two such that neither part's representation crosses the antimeridian.
A rectangle extending from 40 degrees N, 170 degrees E across the antimeridian to 50 degrees N, 170 degrees W should be cut in two and represented as a MultiPolygon. 
   {
       "type": "MultiPolygon",
       "coordinates": [
           [
               [
                   [180.0, 40.0], [180.0, 50.0], [170.0, 50.0],
                   [170.0, 40.0], [180.0, 40.0]
               ]
           ],
           [
               [
                   [-170.0, 40.0], [-170.0, 50.0], [-180.0, 50.0],
                   [-180.0, 40.0], [-170.0, 40.0]
               ]
           ]
       ]
   }

In addition to the library, JSplitPolygon provides a command line utility: 


------------ Help for JSplitPolygon -----------

Usage: java -jar JSplitPolygon.jar -i  [OPTIONS]


with OPTIONS:
  -h|--help                    : This output
  -i|--input         : input file
  -o|--output        : Loads the configuation file
  -l|--level            : debug level : OFF, INFO, DEBUG, TRACE
  -v|--version                 : version
